1. What are props in React?
Answer:

"Props are inputs to React components. They allow data to be passed from parent to child components and are read-only."

2. Can props be changed inside a component?
Answer:

"No, props are immutable. If you want to change data, you should lift the state up to the parent and pass updated props."

3. Difference between Props and State?
